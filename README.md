# 安装
1.下载最新版本的凯恩与林奇2贴图模组文件、
2.下载TexMod材质替换工具。
3.打开TexMod材质替换工具，选择凯恩与林奇2游戏的EXE文件。
4.在封包模式下选择我压缩包中的tpf文件。
5.最后，加载封包即可开始游戏。
